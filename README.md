# [ty.css](http://yukir.net/ty.css/)

out-of-box typography.

## does

* vertical spacing
* sizing

## ignores

* font
* grid system
* ruby
* vertical

use with existing grid systems -- I haven't tested.

## how

    <link rel="stylesheet" href="ty.css">
    <article class="ty"></article>
