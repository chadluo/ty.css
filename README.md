# [ty.css](http://yukir.net/ty.css/)

***

out-of-box typography.

## does

* vertical spacing
* sizing

## ignores

* font
* grid system
* ruby
* vertical

fit with existing grid systems.

## how

    <article class="ty">
    <link rel="stylesheet" href="ty.css" media="screen" type="text/css">
